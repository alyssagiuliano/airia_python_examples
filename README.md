# airia_python_examples

Python examples for working with the Airia platform.

Each example lives in its own folder with a Python file and a README.

## Examples

| Folder | Description |
|---|---|
| [`test_harness/`](test_harness/) | Automated pipeline testing — sends randomised inputs, tracks tool calls, logs results to CSV |
| [`gdrive_upload/`](gdrive_upload/) | Upload a file to Google Drive into a party-name subfolder, creating the folder if it doesn't exist |
